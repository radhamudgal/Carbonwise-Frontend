# 🚀 CarbonWise Dashboard - Quick Start

## Installation & Run (2 Commands)

```bash
npm install
npm start
```

That's it! Your app will open at **http://localhost:3000**

---

## 📦 Package Info

**Recharts** is already in `package.json` (v3.7.0) - no separate installation needed!

---


---

## 🎯 How to Use

1. **Open the app** - You'll see the input panel
2. **Select filters** - State, Year, Sector
3. **Click "Generate Report"** - Charts and insights appear
4. **View results**:
   - Carbon emissions bar chart
   - Sustainability score
   - AI insights
   - Comparison metrics
